Story: Die Ernte-Meisterschaft

In der Welt von Farming Friends findet jedes Jahr die große Ernte-Meisterschaft statt. Farmer aus aller Welt treten an, um den legendären Titel "Meister-Farmer" zu gewinnen.

Du startest als Neuling mit einer kleinen, verlassenen Farm. Dein Ziel: Baue deine Farm auf, meistere Herausforderungen und arbeite dich vom lokalen Amateur bis zur Welt-Meisterschaft hoch!

Story-Progression

PhaseBeschreibungLevel
🌱 TutorialDer alte Meister-Farmer lehrt dich die Grundlagen1-5
🏆 Lokale LigaWettbewerbe gegen Nachbar-Farmen5-15
🥈 Regionale LigaTurniere im Hochtaunuskreis15-30
🥇 Nationale MeisterschaftDie besten Farmer Deutschlands30-50
🌍 Welt-MeisterschaftGlobaler Wettbewerb um den Titel50+

🌱 Maskottchen: Sprouty

Ein kleines, lebendiges Pflänzchen begleitet dich als Helfer und Guide. Sprouty gibt Tipps, feiert deine Erfolge und hilft bei Quests.

Core Gameplay Loop

┌─────────────────────────────────────────────────────┐ │ 1. ANBAUEN │ │ Felder anlegen → Samen pflanzen → Warten │ │ ↓ │ │ 2. ERNTEN │ │ Selbst oder Bots ernten → Rohstoffe sammeln │ │ ↓ │ │ 3. PRODUZIEREN │ │ Rohstoffe → Verarbeitung → Produkte │ │ (Weizen → Mehl → Brot) │ │ ↓ │ │ 4. VERKAUFEN │ │ Warenhaus → NPC-Aufträge → Spieler-Handel │ │ ↓ │ │ 5. INVESTIEREN │ │ Geld → Bessere Tools → Größere Farm → Repeat! │ └─────────────────────────────────────────────────────┘

Offline-Wachstum

Pflanzen und Produktionen laufen auch weiter, wenn der Spieler offline ist. Beim nächsten Login wartet die Ernte!

7

Handel & Wirtschaft

7.1 Warenhaus (Global Trading)

  • Angebote erstellen: "Verkaufe 50× Brot für 2.000 Coins"
  • Anfragen stellen: "Suche 100× Weizen, biete 1.500 Coins"
  • Sofort-Kauf: Andere Spieler-Angebote kaufen
  • Marktgebühr: 5%

7.2 NPC-Aufträge

NPCs erscheinen regelmäßig mit zeitlimitierten Aufträgen:

👨‍🍳 Koch Klaus

"Ich brauche 10× Brot und 5× Eier!"
Belohnung: 500 Coins + 20 XP | Zeit: 2 Stunden

7.3 LKW-Lieferungen

Regelmäßig kommt ein LKW mit Sammelaufträgen. Vollständige Lieferung = Bonus-Belohnung!
